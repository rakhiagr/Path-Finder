## Algorithms

- Dijkstra
- Bellman-Ford
- DFS
- 0-1 BFS
- A\* with manhatten distance heuristic

## Build Guide

```bash
git clone https://github.com/rakhiagr/path-finder
cd path-finder
yarn install
yarn start
```

```
Dependencies Installed -
npm install node-sass@4.14.0
```
